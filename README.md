# Data Science Projects

A collection of 10 end-to-end data science projects covering regression, classification, clustering, NLP, computer vision, time series, and recommender systems.

| # | Project | Category | Technique |
|---|---------|----------|-----------|
| 1 | [Customer Segmentation for Retail Store](1_Customer_Segmentation_for_Retail_Store/README.md) | Clustering | K-Means, PCA |
| 2 | [Movie Rating Prediction](2_Movie_Rating_Prediction/README.md) | Recommender System | SVD Collaborative Filtering |
| 3 | [House Price Prediction](3_House_Price_Prediction/README.md) | Regression | XGBoost, Random Forest, Linear Regression |
| 4 | [Titanic Survival Prediction](4_Titanic_Survival_Prediction/README.md) | Classification | Logistic Regression, Decision Tree |
| 5 | [Sentiment Analysis](5_Sentiment_Analysis/README.md) | NLP | Logistic Regression, Naive Bayes, TF-IDF |
| 6 | [Fraud Detection](6_Fraud_Detection/README.md) | Classification | Random Forest, SMOTE |
| 7 | [Walmart Sales Forecasting](7_Wallmart_Sale_Forecasting/README.md) | Time Series | SARIMA, Prophet, LSTM |
| 8 | [Churn Prediction](8_Churn_Prediction/README.md) | Classification | Logistic Regression, Random Forest, XGBoost |
| 9 | [Fake News Detection](9_Fake_News_Detection/README.md) | NLP | Logistic Regression, TF-IDF |
| 10 | [Fashion CNN](10_Fashion_CNN/README.md) | Computer Vision | ResNet18, Transfer Learning |

## Tech Stack

Python, Pandas, Scikit-learn, PyTorch, TensorFlow/Keras, NLTK, Statsmodels, Streamlit
